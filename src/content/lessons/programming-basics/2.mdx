# 変数とデータ型

## 🎯 学習目標
- 変数の概念を理解する
- 基本的なデータ型を学習する
- 変数の宣言と使用方法を身につける

## 変数とは

**変数**は、データを一時的に保存するための「箱」のようなものです。

この箱には：
- **名前**（変数名）をつけることができる
- **値**を入れることができる
- **値を変更**することができる

## 変数の例（日常生活）

例えば、買い物の場面を考えてみましょう：

```
りんごの値段 = 100円
みかんの値段 = 80円
合計金額 = りんごの値段 + みかんの値段
```

ここで「りんごの値段」「みかんの値段」「合計金額」が変数です。

## プログラミングでの変数宣言

JavaScriptでの変数宣言方法：

```javascript
// 変数の宣言と初期化
let applePrice = 100;
let orangePrice = 80;

// 計算
let totalPrice = applePrice + orangePrice;

// 結果の表示
console.log("りんごの値段: " + applePrice + "円");
console.log("みかんの値段: " + orangePrice + "円");
console.log("合計金額: " + totalPrice + "円");
```

## 基本的なデータ型

プログラミングでは、データの種類によって**データ型**が決まります：

### 1. 数値型（Number）
整数や小数点を含む数値
```javascript
let age = 15;          // 整数
let height = 165.5;    // 小数
let temperature = -5;  // 負の数
```

### 2. 文字列型（String）
文字や文字列
```javascript
let name = "田中太郎";
let message = "こんにちは！";
let empty = "";        // 空の文字列
```

### 3. 真偽値型（Boolean）
真（true）または偽（false）
```javascript
let isStudent = true;
let isRaining = false;
```

## 実際に試してみよう

```javascript
// 自分の情報を変数に入れてみよう
let myName = "あなたの名前";
let myAge = 16;
let isHighSchoolStudent = true;

// 自己紹介文を作ってみよう
let introduction = "私の名前は" + myName + "です。";
introduction = introduction + "年齢は" + myAge + "歳です。";

if (isHighSchoolStudent) {
    introduction = introduction + "高校生です。";
} else {
    introduction = introduction + "高校生ではありません。";
}

console.log(introduction);

// 計算もしてみよう
let birthYear = 2024 - myAge;
console.log("生まれた年: " + birthYear + "年");
```

このコードを実際に実行してみましょう！変数の値を変更して、自分の情報に合わせてカスタマイズしてみてください。

## 💡 重要なポイント

### 変数名のルール
- **英数字とアンダースコア**が使用可能
- **数字から始まってはいけない**
- **予約語**（if、for、letなど）は使用不可
- **意味のある名前**をつける

良い例：
```javascript
let studentCount = 30;
let maxScore = 100;
let userName = "user001";
```

悪い例：
```javascript
let a = 30;        // 何の値かわからない
let 1count = 30;   // 数字から始まっている
let let = 30;      // 予約語を使用
```
