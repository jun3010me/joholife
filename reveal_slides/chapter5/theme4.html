乱数とその生成

乱数の重要性

確率的モデルに不可欠な要素

一様乱数

定義: [0,1]区間で等確率に分布

性質: 独立性、等分布性

用途: 他の分布の生成基盤

線形合同法

X(n+1) = (a × X(n) + c) mod m

a: 乗数, c: 加数, m: 法

適切なパラメータ選択が重要

// 線形合同法の例 long seed = 12345; long a = 1664525; long c = 1013904223; long m = 4294967296; double random() { seed = (a * seed + c) % m; return seed / (double)m; }

疑似乱数

コンピュータの乱数は疑似乱数（決定的アルゴリズムによる）

各種確率分布の生成

逆変換法
1. 一様乱数Uを生成
関数適用
2. 累積分布関数の逆関数
目的分布
3. F⁻¹(U)が目的の分布

指数分布の生成

X = -ln(U) / λ

U: [0,1]の一様乱数

λ: パラメータ（レート）

X: 指数分布に従う乱数

正規分布の生成（Box-Muller法）

X = √(-2ln(U₁)) × cos(2πU₂)
Y = √(-2ln(U₁)) × sin(2πU₂)

U₁, U₂: 独立な一様乱数

X, Y: 標準正規分布に従う

import random def estimate_pi(n): count = 0 for i in range(n): x = random.random() y = random.random() if x*x + y*y <= 1: count += 1 return 4.0 * count / n # 試行回数を変えて精度を確認 for n in [1000, 10000, 100000]: pi_est = estimate_pi(n) error = abs(pi_est - 3.14159) print(f"N={n}: π≈{pi_est:.5f}, 誤差={error:.5f}")
