構造化プログラミングの基本原理

これらの組み合わせで、あらゆるプログラムを構築できる

順次構造

Sequential Structure

順次構造 (Sequential Structure)

プログラムの命令が上から下へ順番に実行される基本的な制御構造。

処理1
name = input()
↓
処理2
age = int(input())
↓
処理3
print(f"こんにちは")

順次構造の例

# 順次構造の例
name = input("お名前を入力してください: ")
age = int(input("年齢を入力してください: "))
print(f"こんにちは、{name}さん")
print(f"あなたは{age}歳ですね")
📝

特徴

最も基本的な処理の流れ

📏

線形処理

条件分岐や繰り返しがない

🎯

予測可能

実行順序が明確

選択構造

Selection Structure

選択構造 (Selection Structure)

条件式の真偽によって実行する処理を選択する制御構造。分岐処理とも呼ばれる。

条件判定
score >= 60
↙️↘️
真の場合
print("合格")
偽の場合
print("不合格")

if文の基本形

❓

単純なif文

score = 85
if score >= 60:
    print("合格です")
🔀

if-else文

score = 45
if score >= 60:
    print("合格です")
else:
    print("不合格です")
🎯

条件式

真（True）または偽（False）の値を返す式

複数条件の判定 (elif文)

score = 92

if score >= 90:
    print("秀")
elif score >= 80:
    print("優")
elif score >= 70:
    print("良")
elif score >= 60:
    print("可")
else:
    print("不可")

elif文

「else if」の略。複数の条件を順次判定する際に使用する構文。

論理演算子

➕

AND演算子 (and)

age = 20
license = True

if age >= 18 and license:
    print("運転できます")
🔀

OR演算子 (or)

weather = "雨"
if weather == "雨" or weather == "雪":
    print("傘が必要です")
🚫

NOT演算子 (not)

online = False
if not online:
    print("オフラインです")

反復構造

Iteration Structure

反復構造 (Iteration Structure)

同じ処理を条件が満たされる間、繰り返し実行する制御構造。ループ処理とも呼ばれる。

🔄

for文

決まった回数や範囲での繰り返し

⏰

while文

条件が真の間の繰り返し

⚡

効率性

同じコードを何度も書く必要がない

for文による反復

🔢

基本的なfor文

# 1から5まで表示
for i in range(1, 6):
    print(f"数値: {i}")
📋

リストの処理

fruits = ["りんご", "バナナ", "オレンジ"]
for fruit in fruits:
    print(f"好きな果物: {fruit}")
📊

range関数

指定した範囲の数値を生成する組み込み関数

while文による反復

🔄

基本的なwhile文

count = 1
while count <= 5:
    print(f"カウント: {count}")
    count += 1  # インクリメント
⚠️

無限ループの防止

while True:
    age = input("年齢を入力: ")
    if age.isdigit() and 0 <= int(age) <= 120:
        break  # ループを抜ける
    print("正しい年齢を入力してください")
⬆️

インクリメント

変数の値を1増加させること。count += 1

ループ制御文

🛑

break文

# ループを強制終了
for i in range(10):
    if i == 5:
        break  # i=5でループ終了
    print(i)

ループを強制的に終了させる

⏭️

continue文

# 現在の反復をスキップ
for i in range(5):
    if i == 2:
        continue  # i=2をスキップ
    print(i)

現在の反復をスキップし、次の反復に進む

🎯

制御の流れ

プログラムの実行を細かく制御

入れ子構造

Nested Structure

入れ子構造 (Nested Structure)

制御構造の中に別の制御構造を記述する構造。ネスト構造とも呼ばれる。

🔗

if文の入れ子

score = 85
attendance = 90

if score >= 60:
    if attendance >= 80:
        print("合格")
    else:
        print("出席不足")
else:
    print("点数不足")
🔄

ループの入れ子

# 九九表
for i in range(1, 10):
    for j in range(1, 10):
        print(f"{i} × {j} = {i*j}")
