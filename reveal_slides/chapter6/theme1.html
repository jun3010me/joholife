What is an Algorithm?

アルゴリズムの定義

問題を解決するための明確で有限な手順。コンピュータが実行できる形で表現された処理の流れ。

🎯

明確性

各ステップが曖昧さなく定義されている

⏹️

有限性

必ず終了する処理の流れ

🔄

再現性

誰が実行しても同じ結果が得られる

機械語

Machine Language

機械語 (Machine Language)

コンピュータのCPUが直接実行できる0と1で表現されたプログラム言語。

🔢

バイナリコード

0と1の組み合わせのみで表現

⚡

直接実行

😰

理解困難

高級言語

High-Level Language

高級言語 (High-Level Language)

人間が理解しやすい形で記述できるプログラミング言語。機械語から抽象化されている。

言語 特徴 主な用途
C言語 システムレベル OS、組み込みシステム
Java オブジェクト指向 企業システム、Android
Python シンプルで読みやすい データ分析、AI、Web
