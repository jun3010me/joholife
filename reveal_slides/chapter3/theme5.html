CPUの性能指標

性能の計算式

処理時間 = 命令数 × CPI ÷ クロック周波数

⏱️

クロック周波数

1秒間の動作回数（Hz）

現在：数GHz

高いほど高速（単純比較は困難）

🔄

CPI

Cycles Per Instruction

1命令あたりの実行サイクル数

命令の複雑さにより変動

🧮

IPC

Instructions Per Cycle

1サイクルあたりの命令数

並列処理能力の指標

例：性能比較

CPU A: 3.0GHz、CPI=2 → 1.5GIPS

CPU B: 2.5GHz、CPI=1.5 → 1.67GIPS

クロック周波数だけでは判断できない

性能計算の実習

課題：CPU性能比較

以下の2つのCPUの理論性能を比較しなさい

CPU A: 3.0GHz、4コア、CPI=2

CPU B: 2.5GHz、6コア、CPI=1.5

1コアあたりの性能：

CPU A: 3.0GHz ÷ 2 = 1.5GIPS

CPU B: 2.5GHz ÷ 1.5 = 1.67GIPS

総合性能：

CPU A: 1.5 × 4 = 6.0GIPS

CPU B: 1.67 × 6 = 10.0GIPS

結論：CPU Bの方が高性能
