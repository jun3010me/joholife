【問題1】
以下のPythonコードを実行すると、どんな値が出力されますか？

x = 0
for i in range(5):
    x = x + 3
print(x)

答え: 15


【問題2】

total = 0
for i in range(2, 7):
    total = total + i

答え: 20


【問題3】
以下のPythonコードを実行すると、どんな値が出力されますか？

numbers = [4, 6, 1, 8]
result = 0
for n in numbers:
    result = result + n
print(result)

答え: 19

【問題5】
以下のPythonコードを実行すると、どんな値が出力されますか？

count = 0
for i in range(1, 16):
    if i % 3 == 0:
        count = count + 1
print(count)

答え: 5
